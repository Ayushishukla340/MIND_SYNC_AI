# 🧠 MindSync – AI Mental Wellness Companion

MindSync is a simple AI-powered mental wellness web app that helps users reflect on their emotions and receive supportive responses through an interactive chat interface.

---

## 🚀 Features

- 😊 Mood selection using emojis  
- 💬 AI-like chat interaction  
- 🧠 Emotion-based responses  
- 🎨 Clean and modern UI  
- ⚡ Fast and lightweight (HTML, CSS, JS)

---

## 🧩 How It Works

1. User selects their current mood (Happy, Sad, Stressed, etc.)
2. The app asks a relevant question based on the selected emotion
3. User types their thoughts
4. The system responds with supportive messages (AI-simulated)

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 🎯 Purpose

This project demonstrates how AI-based emotional support systems can be designed using simple frontend technologies. It simulates an AI therapist experience through rule-based logic and interactive UI.

---

## ⚠️ Note

This is a prototype and does not use real AI or backend processing. Responses are generated using predefined logic.
